hexo 博客添加搜索功能

发表于 2021-05-21 更新于 2021-05-11 分类于 hexo 阅读次数：
安装搜索插件；修改站点配置文件；修改主题配置文件

1、安装搜索插件

在站点目录下 Git bash：npm install hexo-generator-searchdb –save

安装之后，会在站点目录的 public 文件夹下创建一个 search.xml 文件。（没有）

站点目录(根目录)：

可能出现警告：

Warning:

npm WARN optional SKIPPING OPTIONAL DEPENDENCY: fsevents@2.3.2 (node_modules\fsevents):
npm WARN notsup SKIPPING OPTIONAL DEPENDENCY: Unsupported platform for fsevents@2.3.2: wanted {“os”:”darwin”,”arch”:”any”} (current: {“os”:”win32”,”arch”:”x64”})
npm WARN optional SKIPPING OPTIONAL DEPENDENCY: fsevents@1.2.13 (node_modules\hexo-generator-baidu-sitemap\node_modules\fsevents):
npm WARN notsup SKIPPING OPTIONAL DEPENDENCY: Unsupported platform for fsevents@1.2.13: wanted {“os”:”darwin”,”arch”:”any”} (current: {“os”:”win32”,”arch”:”x64”})

原因：fsevents是mac上的相关依赖。

解决：可忽略。或者参考这篇博客操作：npm WARN optional SKIPPING OPTIONAL DEPENDENCY: fsevents@2.1.2 (node_modules\fsevents) -CSDN博客

2、修改站点配置文件 blogs/_config.yml

添加以下：

search:
path: search.json # .xml
field: post
format: html
limit: 10000

  • field：搜索范围，默认是 post，还可以选择 page、all，设置成 all 表示搜索所有页面
  • limit：限制搜索的条目数

网上很多教程都是 path: search.xml ，最后效果是这样的：

效果并不理想…

3、修改主题配置文件 blogs/themes/next/_config.yml

修改local_search为true：

local_search:
enable: true
trigger: auto
top_n_per_article: 1

  • top_n_per_article 字段表示在每篇文章中显示的搜索结果数量，设成 -1 会显示每篇文章的所有搜索结果数量。

最后更新部署到github就可以了。

最好先执行：hexo c (clean)

部署命令：hexo g -d (generate, deploy)

本地预览网页：hexo s (server)
