hexo 博客显示插入图片

发表于 2021-12-16 更新于 2021-12-18 分类于 hexo
本文字数： 428 阅读时长 ≈ 1 分钟
声明：该方法可能并不是最好的，但有用。

安装上传图片插件

在站点目录下 Git bash：npm install hexo-asset-image --save

修改站点配置文件

blog/_config.yml

设置post_asset_folder: true

图片存储位置

hexo-asset-image插件查找图片的位置：**_posts目录下与md文件同名的文件夹。（注意：md文件名中不要出现空格**）

如本博客：

Typora偏好设置

使用如下规则，则插入（或粘贴）图片时Typora自动创建与md文件同名的文件夹，并将图片复制到文件夹内。

注意：如果勾选了自动转义url，图片路径里的中文会变成编码。不过对图片上传无影响。

插入图片的流程

  1. Typora内选择插入图像，选择图片位置；或者复制图片，粘贴到Typora编辑器。（注意：此时Typora已经创建了同名文件夹，并复制了图片。因此不需要再点Typora弹出的复制选项）

  2. 修改图片名称等属性，但是不要改动链接。

    此外，Typora的复制图片的自动过程也可手动完成。
